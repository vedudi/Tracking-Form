# Dijital İş Takip Formu

- Bu proje, kullanıcıların işlerini takip etmelerine ve yönetmelerine olanak sağlayan bir dijital iş takip formunu içerir. Projeyi geliştirirken aşağıdaki kütüphaneleri kullandım

- react-router-dom

- axios

- json-server

- react-toastify

- react-redux

- @reduxjs/toolkit

- uuid

- react-icons

- sass
